
# BackEndNotes

The idea is to create an application in Node.js where the user registers a notes, fills in some information (name, description, note) and creates tags related to it.

# What will be addressed in this challenge:

- Password encryption
- Email validation
- Create an app Node.js
- SQL
- Query Builder

## üîß Installation

### Prerequisites

Before you begin, you will need to have the following tools installed on your machine:
[Node.js] (https://nodejs.org/en/). 
Also, it's good to have an editor to work with the code, such as [VSCode](https://code.visualstudio.com/)

Inside the VsCode terminal write the following code

```
git init
```

```
git clone https://github.com/Gabrielingnau/BackEndNotes
```
Then intade the dependencies with

```
npm install
```

Then just run the project with

```
npm run dev
```
